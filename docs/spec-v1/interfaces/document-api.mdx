The ORD Document API is a plain REST API that is described via an [OpenAPI 3.0 definition](https://spec.openapis.org/oas/v3.0.3).

The OpenAPI yaml file is published here: [/spec-v1/interfaces/DocumentAPI.oas3.yaml](/spec-v1/interfaces/DocumentAPI.oas3.yaml)

It can also be viewed via [Swagger UI](https://pages.github.com/SAP/open-resource-discovery/spec-v1/ord-document-api/).
