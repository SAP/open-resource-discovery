# ORD Interfaces

In this section you will find the technical ORD interfaces.

They are available as human readable documentation and also as machine-readable metadata.

Additionally, the ORD spec is also published as a NPM package, which incluses both the JSON Schema interfaces and generated TypeScrit definitions.

## ORD Version 1

- [ORD Document](../index.md#ord-document)
  - [Human readable documentation](document.md)
  - [JSON Schema](https://pages.github.com/SAP/open-resource-discovery/spec-v1/interfaces/Document.schema.json)
  - [TypeScript Interfaces](https://pages.github.com/SAP/open-resource-discovery/spec-v1/interfaces/Document.ts.txt)
- [ORD Configuration](../index.md#ord-configuration-endpoint)
  - [Human readable documentation](configuration.md)
  - [JSON Schema](https://pages.github.com/SAP/open-resource-discovery/spec-v1/interfaces/Configuration.schema.json)
  - [TypeScript Interfaces](https://pages.github.com/SAP/open-resource-discovery/spec-v1/interfaces/Configuration.ts.txt)
- [ORD Document Endpoint](../index.md#ord-document-endpoint) and [ORD Configuration Endpoint](../index.md#ord-configuration-endpoint)
  - [OpenAPI 3 Definition](./document-api.mdx)
